receivingClientName

receivingClientName: string
  • Defined in src/types.ts:112

A key present in the Sep12Config.customers object. A SEP-12 PUT /customer request will be made with the object contents included in the body, and the id returned will be used in SEP-31 POST /transactions requests as the receiver_id.

sendingAnchorClientSecret

sendingAnchorClientSecret: string
  • Defined in src/types.ts:98

The secret key of the Stellar keypair to use when making requests to the SEP-31 receiving anchor. This is required because only trusted parties (with known Stellar account(s)) should be able to consume a receiving anchor's API.

sendingClientName

  • Defined in src/types.ts:105

Optional transactionFields

transactionFields: any
  • Defined in src/types.ts:121

The fields.transaction object to use in POST /transactions requests. This object will be compared to the fields.transaction object returned by GET /info and is expected to contain a value for every required field.
