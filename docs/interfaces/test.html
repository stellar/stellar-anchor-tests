Optional dependencies

dependencies: DependenciesCaller | Test[]
  • Defined in @stellar/anchor-tests/src/types.ts:319

The Test objects that must run prior to calling this object's Test.run method. runTests and getTests recursively traverse each test's dependency lists until a test with no dependencies is found. Tests that directly or indirectly depend on themselves, creating a cycle, will always fail. If a test expects a value provided by another test, the latter must be included in the former's dependencies array.
